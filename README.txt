This is the main repository associated with the "Self-blinding citizen science to explore psychedelics microdosing" manuscript.
The repository contains:
- the data in csv format (data/), see documentation in the folder
- the analysis scripts required to reproduce all major findings from the paper (scripts/). 

Results related to the accumulative outcomes were generated with SAS, see the corresponding script.
Results related to the acute and post-acute outcomes were analyzed in R, including the stratification analysis.
